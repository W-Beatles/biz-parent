    elastic-job starter模块
3. app-boot-starter-sentry
    sentry starter模块
4. app-facade-common
    通用facade模块
5. app-spring-cloud-archetype
    用于生成spring-cloud项目骨架
6. app-spring-cloud-common

7. app-spring-cloud-test

    spring-cloud starter模块。提供MDC过滤器、控制器/方法切面
